# Socket-Chat-Room
使用 TCP Socket 的多人聊天室！( 計算機網路期末 project )

## Feature
- 使用 python 編寫
- GUI 介面使用 Tkinter 和 PIL(Pillow)
- 使用 TCP Socket
- 支持多 Client 連接 (Multithreading)
- 支持多用戶聊天
- 支持一對一聊天
- 新訊息提示
- 可保存歷史用戶及聊天資訊

## Usage
開啟 server 端
```
python server.py
```
開啟 client 端 ( 可多開 )
```
python client.py
```
